# ArtSync - Digital Art Inspiration Assistant

An AI-powered digital art chatbot that helps artists brainstorm creative ideas, generate detailed prompts, and find reference images.

## Features

- **AI Art Director Chat** — Conversational AI powered by Google Gemini API
- **Reference Image Search** — Pulls real artwork from the Metropolitan Museum of Art API
- **Multilingual Support** — Chat in English, Hindi, Spanish, or French
- **Image Analysis** — Upload artwork images for AI-powered style analysis
- **Conversation Memory** — Context-aware responses throughout the session
- **Human-like Typing** — Character-by-character typing animation
- **Sound Effects** — Subtle audio feedback for send/receive actions
- **Premium UI** — Glassmorphic design with smooth animations

## Tech Stack

- HTML5, CSS3, JavaScript (Vanilla)
- Google Gemini API (Generative AI)
- Metropolitan Museum of Art API (Reference Images)
- Marked.js (Markdown Rendering)

## How to Run

1. Open `index.html` in your browser, or serve with a local server:
   ```
   python -m http.server 8000
   ```
2. Enter your [Gemini API Key](https://aistudio.google.com/app/apikey) or type `demo` for demo mode
3. Start chatting about your art ideas!
